Mode d'emploi EDPC Manager
Simple · Ludique · Compréhensible par tous · IDF 2026
1
🚀 Démarrer
🌐
Ouvrez le lien dans votre navigateur

Chrome, Edge, Firefox. Sur ordinateur ou téléphone.

🏢
Cliquez sur votre CRC (GO, PGE ou LST)

Les managers et conseillers de votre CRC se chargent automatiquement.

👤
Choisissez manager + conseiller + période

T1/T2/T3/T4 ou Point RVI. L'email du manager se remplit tout seul.

🎬
Mode démonstration disponible

Pour tester sans risque — rien n'est sauvegardé, tout disparaît à la fermeture.

💾 Brouillon sauvegardé automatiquement — si vous fermez l'onglet par accident, votre saisie est là à la réouverture.
2
🎯 Priorités identifiées — 1 clic = 2 champs remplis
1
Cliquez sur une priorité

Ex : "Vente sur rebond" → une petite fenêtre s'ouvre.

2
Saisissez le taux et l'appréciation

Tapez "14" → le % s'ajoute tout seul. Choisissez parmi les chips proposées.

✓
Valider → Priorité + Analyse Globale remplis en même temps

Zéro double saisie.

🔵
Conquête
Rebond, Sowee, Izi, Homiris, GAZ, OM, Assurances, Qualité
🟣
Compétitivité
IPAG, Prod BO, Prod FO
🟢
Engagement
Sat FO, Sat BO, Taux de rebonds
⭐
CCX
Missionnements — s'insère aussi dans l'Analyse Globale
💡 Bouton ↩ Annuler pour effacer la dernière ligne. Écriture libre possible dans le champ texte.

5
📋 Plan d'Actions et de Coaching (PAC)
📋
Choisissez une action type

Double écoute, coaching CP, marguerite, suivi hebdo…

📝
Panneau orange → remplissez les valeurs

Date (calendrier), taux, progression. Prévisualisation en temps réel.

✓
Insérer → phrase complète sans placeholder

Aucun [X%] restant à la main.

💡 L'engagement manager

Le champ en dessous du PAC — coaching hebdo, double écoute avant le [date]… C'est ce qui ancre le suivi.

❓ Questions fréquentes
Si je ferme l'onglet, je perds tout ?
Non. Brouillon sauvegardé automatiquement. Tout est là à la réouverture.
Le lien change à chaque mise à jour ?
Non. Le lien est toujours le même. Cliquez "↻ Actualiser" si vous voyez une ancienne version.
Mes traçages sont visibles par d'autres managers ?
Non. Données stockées dans votre navigateur uniquement.
La démo enregistre-t-elle des données ?
Non. Tout disparaît à la fermeture. Rien ne remonte dans les tableaux de suivi.
Je peux l'utiliser sur téléphone ?
Oui. Sur ordinateur c'est plus confortable pour la saisie.
